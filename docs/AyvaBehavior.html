# insertMove(…moves)

Add a move to the front of the action queue.

Parameters:
Name Type Attributes Description
moves Object <repeatable>

objects that describe the moves per the Motion API.

View Source behaviors/ayva-behavior.js, line 178

Example
class ExampleBehavior extends AyvaBehavior {
  generateActions (ayva) {
    this.insertMove({
      to: 0,
      speed: 1,
    },{
      axis: 'twist',
      to: 0,
      speed: 1
    });
  }
}
